На сайт
Вход
Используется Firebase Auth (email + password). Пароль хранится только у Firebase.
Первый раз: создайте пользователя в Firebase Console → Authentication → Users (или включите Email/Password и зарегистрируйтесь).
Резюме (Markdown)
Сохранится в Firestore: документ public/resume.
Важно: Firestore Rules должны разрешать запись только авторизованному пользователю.
